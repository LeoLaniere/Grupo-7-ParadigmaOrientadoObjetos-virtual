2. Desafío de Memoria: Tipos Primitivos vs. Tipos de Referencia

lint : Primitivo
String : Referencia
doublé : Primitivo
boolean : Primitivo
Alumno (Clase personalizada) : Referencia
Integer (Wrapper class) : Referencia
char : Primitivo
int[] (Arreglo de enteros) : Referencia 
String[] (Arreglo de Strings) : Referencia 
Object : Referencia

Primitivos: 
	- Almacenan el valor real exacto.
	- Se encuentran en el Stack.
	- No tienen metodos, son datos puros.
Referencia:
	- Son una direccion de memoria.
	- La variable se guarda en el stack, el objeto en el heap.
	- Tienen atributos y metodos.
